# Consumo de API Meteorológica con Streamlit

Aplicación desarrollada en Python utilizando Streamlit para consumir la API pública Open-Meteo.

## Funcionalidades

- Consulta del clima por ciudad.
- Temperatura actual.
- Velocidad del viento.
- Gráficos de temperatura.
- Gráficos de humedad.
- Gráficos de velocidad del viento.
- Historial de consultas usando SQLite.

## Tecnologías

- Python
- Streamlit
- Pandas
- Requests
- SQLite

## Instalación

```bash
pip install -r requirements.txt
```

## Ejecutar

```bash
streamlit run app.py
```

API utilizada:

https://open-meteo.com/
